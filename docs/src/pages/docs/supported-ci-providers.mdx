# Supported CI Providers

The following CI Providers are officially supported by Bundle Analyzer. For those not in the list, a git fallback is used, it should work everywhere.

- [AppVeyor](https://www.appveyor.com/)
- [Azure Pipelines](https://azure.microsoft.com/en-us/services/devops/pipelines/) with Git repositories
- [Bamboo](https://www.atlassian.com/software/bamboo)
- [Bitrise](https://www.bitrise.io/)
- [Buildbot](http://buildbot.net/)
- [Buildkite](https://buildkite.com/)
- [CircleCI](https://circleci.com/)
- [Codeship](https://codeship.com/)
- [Codefresh](https://codefresh.io/)
- [Drone](https://drone.io/)
- [GitLab CI](https://about.gitlab.com/gitlab-ci/)
- [Greenhouse](https://greenhouseci.com/)
- [Jenkins](https://jenkins.io/)
- [Netlify](https://netlify.com)
- [Semaphore](https://semaphoreci.com/)
- [Shippable](https://app.shippable.com/)
- [TeamCity CI](https://www.jetbrains.com/teamcity/)
- [Travis CI](https://travis-ci.org/)
- [Wercker](http://wercker.com/)
